# desafio-hdi
## ejecucion
Para utilizar por favor ejecutar docker compose up y el compose se encargara de crear el contenedor y exponer el puerto 8080

## ejecucion debug
Para ejecutar en modo debug en visual studio code, solo es necesario crear un ambiente venv, e instalar las librerias descritas en **requirements.txt** con pip
luego en vscode entrar en modo **run and debug** y seleccionar el metodo de fastAPI
